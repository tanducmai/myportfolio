Problems With Traditional And Physical Data Storage

  1. Limited by physical size
  2. Limited multi-access - a borrowed book can only be viewed by one person at a time.
  3. Limited search methods (title, author, subject)
  4. Slow turn-around
  5. Multiple indexes that must be rigorously maintained.
  6. Complex Query Limitations - vast amounts of time wasted counting the number of books on a subject or by an author.
  7. Anomalies: - What IFs:
  • A card is misplaced?
  • A catalogue card is misplaced?
  • An item is entered into / deleted in one index but not the others?

Source: Britannica

Modern And Digital Data Storage

Data can now be stored in files, such as text files, Excel sheets, Word documents.

Source: Orange Matter

Advantages
  1. Low cost
  2. No extra software / hardware is needed
Disadvantages
  1. Size - the larger the volume of data, the slower the retrieval of them (yet still better than physical data).
  2. Update / Synchronisation - files are hard to share. Updates from multiple users often end up with multiple versions of the same file or even overwriting someone’s changes!
  3. Accuracy - duplication is wasteful and can cause confusion (inconsistent data).
  4. Security - restriction to some fields is more difficult to manage (e.g., Tax File Numbers, IDs + usernames).
